## Структура проекта

```
app/
├── types/           # TypeScript типы
├── constants/       # Константы и mock данные
├── hooks/           # React хуки
├── utils/           # Утилиты и обработчики
├── components/
│   ├── shared/      # Общие компоненты
│   ├── client/      # Клиентское приложение
│   ├── admin/       # Админ-панель
│   └── tech/        # Приложение механика
├── page.tsx         # Оригинал (сохранен)
└── page-new.tsx     # Новая модульная версия
```

## Как использовать

### Запуск приложения
```bash
npm run dev
```
Приложение доступно на http://localhost:3000

### Переключение на новую версию
```bash
# 1. Переименовать файлы
mv app/page.tsx app/page-old.tsx
mv app/page-new.tsx app/page.tsx

# 2. Перезапустить сервер
npm run dev
```

### Откат к оригиналу
```bash
mv app/page.tsx app/page-new.tsx
mv app/page-old.tsx app/page.tsx
```
